# Wireframes Iniciales SMT

## 1. Splash

Logo SMT  
Nombre: Sistema Mental Tótem  
Frase: Construye la persona que quieres ser  

## 2. Bienvenida

Título: Bienvenido a SMT  
Texto: Tu sistema operativo de desarrollo humano.  
Botón: Comenzar  

## 3. Crear Identidad

Campos:

- Nombre del usuario
- Nombre del Tótem
- Frase personal
- Propósito inicial

## 4. Crear Tótem

El usuario selecciona sus pilares principales:

- Espíritu
- Sabiduría
- Arte
- Atleta
- Liderazgo
- Economía
- Familia
- Comunidad
- Otro

Luego ordena los pilares según prioridad.

## 5. Inicio

Elementos:

- Tótem visual
- Hábitos del día
- Objetivos activos
- Reflexión rápida
- Recomendación IA

## 6. Mi Tótem

Elementos:

- Lista de pilares
- Nivel de cada pilar
- Progreso
- Acciones asociadas

## 7. Diario

Elementos:

- Entrada diaria
- Estado emocional
- Pilar relacionado
- Reflexión IA
