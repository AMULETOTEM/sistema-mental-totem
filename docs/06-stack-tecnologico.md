# Stack Tecnológico SMT

## Aplicación móvil

Framework elegido:

Flutter

Motivo:

Permite desarrollar una sola aplicación para Android e iOS usando una misma base de código.

## Backend

Backend elegido:

Supabase

Motivo:

Incluye base de datos PostgreSQL, autenticación, almacenamiento y funciones backend.

## Base de datos

Motor:

PostgreSQL

Tablas iniciales:

- users
- profiles
- totems
- pillars
- habits
- goals
- journal_entries
- coherence_scores
- ai_conversations

## Inteligencia Artificial

Proveedor inicial:

OpenAI API

Uso previsto:

- Reflexiones personalizadas
- Análisis del diario
- Recomendaciones de hábitos
- Análisis del Índice de Coherencia
- Asistente personal SMT

## Arquitectura general

Frontend:

Flutter

Backend:

Supabase

Base de datos:

PostgreSQL

IA:

OpenAI API

Publicación:

Google Play primero.  
App Store después.
